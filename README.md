# Week-3--App-3-Workflow-Wizard

A no-code / low-code AI agent that designs automations for real people.

## Quick Start

Navigate to the `workflow-wiz` directory for the full application:

```bash
cd workflow-wiz
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your API keys
python app/run.py
```

See [workflow-wiz/README.md](workflow-wiz/README.md) for detailed documentation.
